## 📄 Аннотация проекта

**CarShare** — это веб-ориентированная информационная система, предназначенная для автоматизации процессов краткосрочной аренды автомобилей (каршеринг). Проект решает задачу комплексного управления автопарком, включая бронирование, финансовый учет, проверку безопасности клиентов и мониторинг технического состояния транспортных средств.

### 🎯 Цель проекта
Создание безопасной и прозрачной среды для аренды авто, где минимизированы риски для компании (долги, ДТП, недопуск водителей) и максимизировано удобство для клиента (онлайн-бронирование, личный кабинет).

### ❗ Актуальность и проблема
Современные сервисы каршеринга сталкиваются с рядом проблем, которые решает наша система:

*   **Проблема:** Аренда клиентам с просроченными правами.  
    **Решение:** Автоматическая проверка срока действия прав через базу данных.
*   **Проблема:** Клиенты с задолженностями продолжают арендовать.  
    **Решение:** Автоматическая блокировка аккаунта при наличии долгов.
*   **Проблема:** Отсутствие учёта инструктажей по безопасности.  
    **Решение:** Журнал инструктажей с датой истечения и напоминаниями.
*   **Проблема:** Сложность отслеживания состояния автопарка.  
    **Решение:** Дашборд с реальными статусами всех автомобилей.
*   **Проблема:** Ручной расчёт стоимости и штрафов.  
    **Решение:** Автоматизированная финансовая система с прозрачной историей.

## ⚙️ Ключевой функционал

### Для клиентов
*   📋 **Регистрация и авторизация:** Создание аккаунта, вход по логину/паролю.
*   🚘 **Каталог автомобилей:** Просмотр доступных машин с фото и характеристиками.
*   📅 **Бронирование:** Выбор дат и времени аренды, онлайн-оплата.
*   👤 **Личный кабинет:** История аренд, текущие статусы, документы.
*   ⚠️ **Уведомления:** О статусе брони, сроке действия прав и инструктажей.

### Для менеджеров
*   📝 **Выдача автомобиля:** Проверка клиента, подтверждение выдачи.
*   🔍 **Осмотр при возврате:** Фиксация повреждений, начисление штрафов.
*   📚 **Проведение инструктажа:** Отметка о прохождении техники безопасности.
*   💳 **Управление платежами:** Контроль депозитов, погашение задолженностей.

### Для администрации
*   📊 **Дашборд рисков:** Визуализация проблемных клиентов и автомобилей.
*   📈 **Аналитика прибыли:** Отчёты по доходам, загрузке парка, популярности моделей.
*   👥 **Управление пользователями:** Блокировка/разблокировка клиентов, редактирование данных.
*   🚗 **Управление автопарком:** Добавление/списание автомобилей, изменение тарифов.

## 🗄️ Структура базы данных

Основные таблицы проекта:

*   **clients:** Данные клиентов (id, ФИО, права, телефон, статус блокировки).
*   **cars:** Автопарк (id, марка, модель, госномер, статус, тарифы).
*   **rentals:** Договоры аренды (id, client_id, car_id, даты, стоимость, статус).
*   **payments:** Финансовые операции (id, rental_id, сумма, тип, статус оплаты).
*   **safety_briefings:** Инструктажи (id, client_id, дата, тип, срок действия).
*   **users:** Учётные записи (id, login, password_hash, role).
