# 🚗 CarShare: Система управления каршерингом

> **Тип проекта:** Учебный лабораторный проект (Web-приложение)  
> **Год реализации:** 2026

---

## 📄 Аннотация проекта

**CarShare** — это веб-ориентированная информационная система, предназначенная для автоматизации процессов краткосрочной аренды автомобилей (каршеринг). Проект решает задачу комплексного управления автопарком, включая бронирование, финансовый учет, проверку безопасности клиентов и мониторинг технического состояния транспортных средств.

### 🎯 Цель проекта
Создание безопасной и прозрачной среды для аренды авто, где минимизированы риски для компании (долги, ДТП, недопуск водителей) и максимизировано удобство для клиента (онлайн-бронирование, личный кабинет).

### ⚙️ Ключевой функционал
| Модуль | Возможности |
| :--- | :--- |
| **🚘 Автопарк** | Учет машин, статусы (свободна/ремонт/аренда), фото, характеристики |
| **📅 Бронирование** | Выбор дат, проверка доступности, автоматический расчет стоимости |
| **🛡 Безопасность** | Проверка водительских прав, учет инструктажей, блокировка нарушителей |
| **💰 Финансы** | Управление депозитами, штрафы, история платежей, контроль долгов |
| **📊 Аналитика** | Дашборд для руководства (прибыль, риски, загрузка парка) |

### 🛠 Технологический стек
*   **Backend:** Python, Flask, SQLAlchemy
*   **Frontend:** React, Material UI, Axios
*   **Database:** PostgreSQL
*   **Tools:** Git, Docker, Swagger
